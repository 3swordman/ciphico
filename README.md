## 介绍

Phico 的一个实现

## 什么是 phico?

Phico 是一个编程语言，所有人都可以轻松地使用它。

## 什么是 ciphico?

Ciphico 是一个非常快的 phico 的实现。
Ciphico 包含一个解释器与一个编译器。

## 安装方式

1. 下载源文件
2. ```mkdir build```
3. ```cd build```
4. ```cmake ..```
5. ```cmake --build . --config Release```
6. 在linux上，使用 ```./ciphico [source file]```，在 Windows 上，使用 ```ciphico [source file]```
